Voici certainement l'une des « astuces » les plus basiques : comment installer git ! Et bien, c'est assez simple.

Si les informations données ici venaient à être périmées, n'hésitez pas à le signaler en commentaire.

p=. "!http://farm4.static.flickr.com/3047/2754478731_6cac6d30a8_m.jpg!":http://flickr.com/photos/15708236@N07/2754478731/

*Windows*

git est disponible sous Windows ! Le meilleur guide à suivre est en anglais : "Illustrated Guide to Git on Windows":http://nathanj.github.com/gitguide/ par "nathanj's":http://github.com/nathanjis. Il propose en plus une partie sur les bases de git : "Git on MSys":http://code.google.com/p/msysgit/. Une autre option consiste à utiliser Cygwin : "compiling from source":http://techblogging.wordpress.com/2008/04/11/compiling-and-installing-git-on-windows-under-cygwin/ using "Cygwin":http://www.cygwin.com/.

*OSX*

Sous 10.4 (Tiger), installez "MacPorts":http://macports.org et lancez @sudo port install git-core@. Il reste possible de compiler git soi-même en cas de problème : ":http://larrytheliquid.com/2007/12/29/compiling-git-and-git-svn-on-osx-tiger/

Sous 10.5 (Leopard), c'est encore plus simple, du fait de l'existence du "Git OSX Installer":http://code.google.com/p/git-osx-installer/.

*Linux*

Comme d'habitude :

p=. @apt-get install git-core@
ou
@yum install git-core@
ou une autre variante, selon votre distribution.

À noter, l'"excellent tutorial pour compiler git sous Ubuntu":http://chrisolsen.org/2008/03/10/installing-git-on-ubuntu/. par "Chris Olsen":http://chrisolsen.org.

Deux sources complémentaires d'information : le "Git Community Book":http://book.git-scm.com/2_installing_git.html et le "GitWiki":http://git.or.cz/gitwiki/Installation!
